・環境

必須環境

OS Windows XP　以降
Visual Studio 2008　以降　がある
Microsoft DirectX9.0c(August 2009)以降のSDK

推奨環境

CPU IntelCore2 Duo 1.40GHz　以上
メモリ　数MB

・機能（*が付いているものは未実装)

  • Win32API関連

    • ウインドウ作成（位置、サイズ、タイトル、アイコン、カーソル）
    • メッセージループ処理


  • DirectX関連

    • グラフィック
      • 2D関連
        • 画像ファイル(gif, jpg, png, bmp, dds)の読み込み
        • 画像の描画（座標、反転、回転、拡縮）
        • 画像のアルファブレンド（半透明、加算、減算、乗算）


      • 3D関連
        • *モデルファイル(x)の読み込み
        • *モデルの描画（座標、反転、回転、拡縮）
        • *テクスチャの貼付け
        • *アルファブレンド（半透明、加算、減算、乗算）
        • *カメラ操作
        • *ライティング
        • *移動アニメーション


      • その他
        • 初期化（ウインドウ、フルスクリーン切り替え）
        • *簡易図形描画(点、線、矩形、三角形、円）2D3Dどちらにも
        • *簡易文字出力
        • *フォント（読込、位置サイズ指定、ブレンディング）


    • サウンド
      • *音声ファイル（wav, mp3, ogg)の読み込み
      • *再生、一時停止、途中再生、停止、音量


    • 入力
      • *キーボードの押下状態
      • *マウスのクリック、ホイール
      • *ゲームパッド（ボタン、アナログスティック）

  • 数学

    • 3次元ベクトルクラス
    • *4次元行列

  • その他

    • *ゲームシーン管理
    • *FPS管理クラス
    • コンソール画面の作成
    • *CSVファイル読み書き
    • *衝突判定
